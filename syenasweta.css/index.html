syenasweta.css is a CSS (Cascading Style Sheet) for Idless and Classless HTML (Hyper Text Markup Language).

No class. No identity. Those are the two concepts by syenasweta.css.

When everything is identified and/or classified, a partition is born which in the end makes the object not have an equivalent value substance. All HTML elements have special attributes. syenasweta.css does not use global attributes--class and id, instead it uses special HTML attributes with their own characteristics to create styling in HTML layouts.
